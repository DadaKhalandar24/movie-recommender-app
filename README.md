# Movie Recommendation Web Application

A full-stack web application that provides AI-powered movie recommendations based on user preferences.

## Features

- AI-powered movie recommendations using OpenAI API
- User-friendly React frontend with modern UI
- Fastify backend with SQLite database
- Persistent storage of recommendations history
- Responsive design for all devices
- Example queries for quick testing

## Tech Stack

**Frontend:**
- React 18
- Axios for API calls
- CSS3 with modern styling

**Backend:**
- Node.js with Fastify
- OpenAI API integration
- SQLite database
- CORS support

## Prerequisites

1. Node.js (v16 or higher)
2. npm or yarn
3. OpenAI API key
